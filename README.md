# #Model v2:基于v1补充约60张图片训练出的二代模型,于第二轮达到最佳,十七轮早停

###  v2 训练现象分析：EarlyStopping at Epoch 2
**现象**：加载 v1 `best.pt` 微调后，验证指标于 Epoch 2 达到峰值，连续 15 轮未提升触发早停（Epoch 17 终止）。
- 
**归因**：
- 
  1. v1 权重已具备强特征提取能力，v2 数据分布连续，模型仅需 1~2 轮快速适配即进入平台期。
     
  2. 默认学习率（lr0=0.01）对微调偏大，前期梯度跳跃导致 val 指标早期见顶。
     
  3. 验证集规模（25张/36实例）对波动敏感，放大早停触发概率。
     
**决策**：

  - 当前模型 P=0.905 / R=0.946 / mAP50-95=0.761，已满足辅助标注与阶段二部署门槛。

  - 最终版（v3）将调整 `lr0`, `warmup_epochs`, `patience`，使收敛曲线平滑延伸至 20~40 轮，获取更稳定的泛化评估。
 
### 训练指标与曲线分析

**整体表现**：训练运行 17 轮后触发早停（`patience=15`），最佳权重保存于 Epoch 2。验证集最终指标：`Precision=0.88 / Recall=0.99 / mAP50=0.94 / mAP50-95=0.73`。

**Loss 趋势**：`cls_loss` 持续下降，分类头快速适应新数据分布；`box_loss` 与 `dfl_loss` 平稳收敛，边界框回归稳定。验证集 Loss 随训练集同步下降，泛化轨迹健康。

### 使用体验:

对于3和哨兵的分辨能力较弱,在暗光条件下偶发分不清红色和蓝色的问题
